## Root cause

DataInputStream.skipBytes() may return fewer bytes than requested; its return value was discarded.

## Approved fix

Replaced skipBytes(addrLen + 2) with readFully(ByteArray(addrLen + 2)).

**Why this is causal:** readFully either consumes the exact BND.ADDR and BND.PORT length or fails the handshake before TLS can receive protocol bytes.
